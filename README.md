How to set up:
---
Server:
1. Navigate to backend folder:
   `cd ./backend`
2. Install dependencies:
   `npm i`
3. Set secrets:
   In `.env` set the constants `GEMINI_API_KEY`, `OPENAI_API_KEY` and `PROVIDER_TYPE` to select specific model
4. Run the app:
   `npm run dev`
---
Client:
1. Install dependencies:
   `npm i`
2. Run the app:
   `npm run dev`
